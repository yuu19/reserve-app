# 契約と Premium

組織の現在プラン、Premium トライアル、支払い方法、契約履歴を確認する手順です。

契約の開始、支払い方法の登録、プラン変更は organization owner だけが操作できます。admin、manager、staff、participant は契約状態を確認できる場合がありますが、契約変更や支払い設定は実行できません。

## このページでできること

- 組織の現在プランを確認する
- 7日間の Premium トライアルを開始する
- トライアル中に支払い方法を登録する
- Premium 契約中にプラン変更や請求書・領収書を確認する

## 利用前に確認すること

- この手順は管理画面の `契約` 画面を前提にしています。
- 契約は組織ごとに管理されます。教室ごとの契約状態はありません。
- 契約変更と支払い設定は organization owner のみ操作できます。
- 支払い方法、請求先情報、プラン変更は Stripe の画面で扱います。WakuReserve はカード番号などの支払い詳細を保存しません。
- 回数券購入のアプリ内 Stripe 決済は現在利用できません。回数券購入は現地決済または銀行振込の承認フローで扱います。

## 現在プランを確認する手順

1. 管理画面にログインし、サイドメニューの `契約` を開きます。

   ![契約画面](/manuals/admin-contracts-and-premium/01-open-contracts.png)

2. `現在プラン` で、プラン名、契約ティア、請求周期、次回更新日、解約予定を確認します。

3. `管理アクション` を確認します。owner の場合は、現在の契約状態に応じて操作ボタンが表示されます。owner 以外の場合は閲覧専用の案内が表示されます。

## Premium トライアルを開始する手順

同じ組織で開始できる Premium トライアルは1回だけです。トライアル開始時点では、支払い方法はまだ登録されません。

1. `契約` 画面で `7日間のPremiumトライアルを開始` をクリックします。

2. 画面に `プレミアムトライアル` と表示されたら、トライアル開始は完了です。

3. `トライアル終了日` と `支払い方法の登録状況` を確認します。

   ![Premium トライアル中の契約画面](/manuals/admin-contracts-and-premium/02-trial-payment-method.png)

## 支払い方法を登録する手順

トライアル終了後も Premium 機能を継続したい場合は、トライアル期間中に支払い方法を登録します。

1. `支払い方法の登録状況` が未登録であることを確認します。

2. `支払い方法を登録` をクリックします。

3. Stripe の画面へ移動したら、画面の案内に従って支払い方法を登録します。

4. WakuReserve に戻ったら、`支払い方法の登録状況` が更新されているか確認します。反映に時間がかかる場合は、数分後にもう一度確認してください。

## Premium 契約を管理する手順

Premium 契約中のプラン変更や支払い方法の更新は、Stripe Customer Portal で行います。

1. `現在プラン` が `Premiumプラン` であることを確認します。

2. `プランを変更` をクリックします。

3. Stripe Customer Portal が開いたら、プラン変更、支払い方法の更新、解約などを必要に応じて進めます。

   ![Premium プランの契約管理画面](/manuals/admin-contracts-and-premium/03-paid-plan-management.png)

4. Stripe Customer Portal での操作後、WakuReserve の `契約` 画面で反映後の状態を確認します。

## 請求書・領収書を確認する手順

請求書や領収書が発行されている場合、owner は `契約` 画面から参照リンクを確認できます。支払い詳細そのものは WakuReserve には保存されません。

1. `契約履歴` で、契約変更や支払いイベントの履歴を確認します。

2. `請求書・支払いイベント` までスクロールします。

3. 表示されている場合は、`請求書を開く`、`PDFを開く`、`領収書を開く` から Stripe の書類を確認します。

   ![請求書と支払いイベント](/manuals/admin-contracts-and-premium/04-billing-documents.png)

## 注意点

- 支払い遅延、未払い、契約処理中、解約済みの場合は、Premium 機能の利用可否が制限されることがあります。
- 支払い方法の登録後、Stripe 側の更新確認に時間がかかる場合があります。完了状態に変わらない場合は、時間をおいて再確認してください。
- `請求書・支払いイベント` は、表示できる参照がある場合だけ表示されます。未生成、確認中、利用不可の状態では書類リンクが表示されないことがあります。
- Premium トライアルが終了して無料プランに戻っても、組織、教室、予約、参加者などの業務データは削除されません。Premium 機能だけが利用不可になります。

## よくあるつまずき

### 契約操作のボタンが表示されない

契約変更と支払い設定は organization owner のみ操作できます。管理画面を利用できる role でも、owner でない場合は閲覧専用になります。操作が必要な場合は、組織の owner に依頼してください。

### 支払い方法を登録したのに未登録のまま表示される

Stripe 側の登録完了を確認するまで、反映に時間がかかる場合があります。数分後に `契約` 画面を開き直して確認してください。

### 請求書や領収書が表示されない

請求書や領収書がまだ発行されていない、確認中、または利用不可の状態ではリンクが表示されません。必要な書類が見つからない場合は、Stripe Customer Portal の表示も確認してください。

### Premium 機能が使えない

無料プラン、未払い、契約処理中、解約済みなどの状態では Premium 機能を利用できないことがあります。`現在プラン` と画面上の注意表示を確認してください。

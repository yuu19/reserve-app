# 管理者向け初回セットアップ

管理者アカウントを新規登録し、最初の組織を作成して管理画面の利用を始める手順です。

## このページでできること

- 管理者アカウントを新規登録する
- 最初の組織を作成する
- 管理画面のダッシュボードに入る

## 利用前に確認すること

- この手順は `https://web.wakureserve.com/` の管理者向け導線を前提にしています。
- 新規登録では `表示名`、`メールアドレス`、`パスワード` が必要です。
- パスワードは 8 文字以上で入力します。
- `組織 slug` は組織ごとの識別子です。あとで URL に使われます。

## 初回セットアップ手順

1. `https://web.wakureserve.com/admin/login` を開き、`新規登録` タブに切り替えます。

   ![管理画面ログインの新規登録タブ](/manuals/admin-getting-started/01-admin-login-entry.png)

2. `表示名`、`メールアドレス`、`パスワード` を入力して、`新規登録` をクリックします。

3. `最初の組織と教室を作成` 画面が表示されたら、まず入力欄の位置を確認します。

   ![初回セットアップ画面](/manuals/admin-getting-started/02-onboarding-form.png)

4. `組織名` と `組織 slug` を入力します。`初期教室名` と `初期教室 slug` は、あとから設定したい場合は空欄にします。

   ![組織名と組織 slug を入力した初回セットアップ画面](/manuals/admin-getting-started/06-onboarding-clear-classroom-fields.png)

5. `組織と教室を作成` をクリックします。

6. ダッシュボードが表示されたら、初回セットアップは完了です。ここから `設定`、`教室管理`、`予約運用` などへ進めます。

   ![管理画面ダッシュボード](/manuals/admin-getting-started/04-admin-dashboard.png)

## 補足

- `初期教室名` と `初期教室 slug` を空欄にすると、組織名ベースの初期教室が自動で用意され、あとから `教室管理` で見直せます。
- 組織ロゴは任意です。後から設定しても問題ありません。

## 2026-04-23 時点の確認メモ

- `web.wakureserve.com` では、`組織と教室を作成` を押したあとにオンボーディング画面へ留まることがありました。
- 同日時点では、組織自体は作成されており、`/admin/dashboard` を直接開くと管理画面へ進めることを確認しました。
- `初期教室名` と `初期教室 slug` は入力途中の値が自動で入ることがあったため、そのままにせず内容を確認してから送信してください。

## よくあるつまずき

### 新規登録できない

- メールアドレスの形式を確認してください。
- パスワードが 8 文字以上になっているか確認してください。

### 作成後に画面が切り替わらない

- 一度 `https://web.wakureserve.com/admin/dashboard` を開いて、ダッシュボードへ進めるか確認してください。
- ダッシュボードに入れた場合は、`設定` または `教室管理` から内容を見直せます。

### 初期教室の内容を後から変更したい

- ダッシュボードから `教室管理` へ移動し、教室名や slug を更新してください。
